## 1.3 数和表达式

基本的运算  
加 +  
减 -  
乘 * 除 /  
整除 //

乘方 **
乘方的优先级比负（单目减）高

```python
-3 ** 2  # -9
(-3) ** 2  # 9
```

十六进制、八进制和二进制

```python
# 十六进制
print(0xAF)  # 175
# 八进制
print(0o10)  # 8
# 二进制
print(0b1011010010)  # 722
```

## 1.8 模块
```python
# 导入模块
import math
math.ceil(12.5) # 调用模块里的函数

# 直接导入函数，但是可能存在命名冲突
from math import sqrt
sqrt(9) # 直接通过函数名调用

# 除非必须使用from版的import命令，否则应坚持使用 常规版import命令。
```

### 1.8.1 cmath和复数
```python
import cmath
print(cmath.sqrt(-1))  # 1j

# 复数运算
print((1 + 3j) * (9 + 4j))  # (-3+31j)
```
